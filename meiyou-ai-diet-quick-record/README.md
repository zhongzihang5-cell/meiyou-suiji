## 原型调试

页面右下角 **Tweaks** 面板可切换：

- **用户身份**：经期 / 卵泡期 / 孕期 / 育儿
- **进入方式**：直接点 Tab（引导输入页）/ 从分析进来（带红绿灯 + 时间轴）

## 交互流程（V3 — 特色输入页）

**当前迭代重点：语音/文字便捷输入（Portal 页）**

1. **直接点 Tab** → 沉浸式 **Portal 输入页**（动效氛围 + 大标题引导）
2. 底部 **玻璃 Dock**：「语音 | 文字」双模式
   - **语音模式**：中央大麦克风 + 脉冲环 → 点击进入全屏聆听室
   - **文字模式**：多行输入 + 发送
3. 输入后 → **「已记录 XX」** 绿色打勾条
4. 左上角 **「我的随记」** → 收起至日卡时间轴
5. **聆听室**：光球 + 波形 + 「随记正在听…」

Tweaks 可切换四大身份，context / 打勾字段随身份变化。

## 交互流程（V2 概要）

1. **直接点 Tab** → 引导输入页（天气 + 周期上下文 + AI 引导）
2. 输入文字 → 结构化字段打勾动效「已自动记录 月经/心情…」
3. 点击语音按钮 → **沉浸式聆听页**（全屏渐变光球）
4. 关闭引导页 → **精致日卡时间轴**
5. 从分析进来 → 顶部红绿灯 + 日卡时间轴，右下角 ✏️ 可再次打开引导页
